# Conflict Rules

Before any Decision Stance, check whether the idea duplicates or conflicts with another exposure.

## Must Not Be Treated As Independent

```md
EURUSD Long + GBPUSD Long
= duplicated short-USD exposure

EURUSD Short + GBPUSD Short
= duplicated long-USD exposure

XAUUSD Long + EURUSD Long
= often duplicated weak-USD thesis

USDJPY Long + XAUUSD Long
= possible conflict unless justified by separate logic

GBPUSD Long + EURGBP Short
= duplicated GBP exposure

USDCAD Long + bullish oil thesis
= conflict unless CAD weakness is separately justified
```

## Order Conflict Rules

```md
Same Symbol Buy + Same Symbol Sell
= forbidden unless explicitly marked as hedge

Multiple USD-short positions
= must be treated as one USD basket thesis

Multiple USD-long positions
= must be treated as one USD strength thesis

Risk-on position + Risk-off position
= must explain whether hedge or contradiction

If conflict is unresolved:
Decision Stance = WAIT / NO DECISION
```

## Conflict Conclusion Labels

Use one:

- Clean
- Duplicated Risk
- Conflict
- Hedge
- Unclear

## Archive Rule

A Decision Pack must not hide conflict.

If conflict exists, the pack should explain whether the conflict is intentional, a hedge, duplicated risk, or a reason to wait.
