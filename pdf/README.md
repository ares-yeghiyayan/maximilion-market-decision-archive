# PDF Artifacts

PDF files are optional polished exports of Markdown Decision Packs.

The Markdown file is the source of truth.

Recommended order:

1. Write the Decision Pack in Markdown.
2. Review it in GitHub.
3. Export a PDF version only after the Markdown version is stable.
4. Store the PDF in the matching date folder.

## Naming Convention

```text
YYYY-MM-DD_symbol_decision_pack.pdf
```

## Boundary

PDF files must follow the same public-safe rules as Markdown files.
