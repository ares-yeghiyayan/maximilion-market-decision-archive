# Reviews

This folder stores later review notes for Decision Packs.

The goal is to create a memory of decisions under risk.

## Review Windows

- 30-day review
- 60-day review
- 90-day review

## Review Questions

Each review should ask:

- What happened?
- Was the original logic valid?
- Was the risk correctly identified?
- Was the decision stance too aggressive, too defensive, or appropriate?
- What should Maximilion learn?

## Boundary

Reviews should evaluate decision quality, not advertise trading performance.
