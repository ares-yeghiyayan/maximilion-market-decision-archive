# Decision Packs

This folder contains public-safe Market Decision Packs.

Each pack should follow:

```text
framework/DECISION_PACK_TEMPLATE.md
```

## Naming Convention

Use this format:

```text
YYYY-MM-DD_symbol_decision_pack.md
```

Example:

```text
2026-07-01_xauusd_decision_pack.md
```

## Current Rule

Do not add more assets beyond the initial universe until the process is repeatable:

- XAUUSD
- EURUSD
- GBPUSD
- USDJPY
- USDCAD
